# EVE Trade

This Electron application retrieves real-time EVE Online market data and composes lists of items to buy in order to maximize profit for particular trading trip.

## Project setup

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn serve
```

### Compiles and minifies for production

```
yarn build
```

### Lints and fixes files

```
yarn lint
```
